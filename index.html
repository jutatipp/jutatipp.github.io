สวัสดี ฉันคือ จุฑาทิพย์

นักศึกษาสาขาวิทยาการคอมพิวเตอร์สารสนเทศ มีความสนใจด้านการออกแบบเว็บไซต์ที่ทันสมัย และการพัฒนา Web Application ให้ใช้งานง่าย ดูเป็นมืออาชีพ และตอบโจทย์ผู้ใช้งานจริง

เกี่ยวกับฉัน

ประวัติการศึกษา

กำลังศึกษาระดับปริญญาตรี สาขาวิทยาการคอมพิวเตอร์สารสนเทศ มหาวิทยาลัยขอนแก่น

ทักษะความสามารถ

สามารถพัฒนาเว็บไซต์ด้วย HTML, CSS, JavaScript รวมถึงออกแบบ UI ให้ดูทันสมัยและใช้งานง่าย

เป้าหมาย

ต้องการพัฒนาทักษะด้าน Frontend และ UX/UI เพื่อก้าวสู่การเป็นนักพัฒนาเว็บไซต์มืออาชีพ

ผลงาน

เครื่องคิดเลขออนไลน์
